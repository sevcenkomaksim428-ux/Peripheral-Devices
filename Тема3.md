# Тема 3. Порівняльна характеристика та огляд оперативної пам'яті DDR4 та DDR5

## 1. Характеристики оперативної пам'яті типів DDR4 та DDR5

Оперативна пам'ять (RAM) є критично важливим компонентом для швидкодії системи. Наразі найпоширенішими є два стандарти: DDR4, який домінував на ринку протягом останнього десятиліття, та новітній стандарт DDR5, який забезпечує значний приріст продуктивності.

### Основні характеристики DDR4
* **Частота:** Зазвичай від 2133 МГц до 3200 МГц (JEDEC), з розгоном (XMP) до 4000-5000 МГц.
* **Напруга:** Стандартна напруга — 1.2В.
* **Архітектура:** Один 64-бітний канал даних на модуль.
* **Керування живленням:** Розміщено на материнській платі.

### Основні характеристики DDR5
* **Частота:** Базова від 4800 МГц, з можливістю розгону до 6400-8000+ МГц.
* **Напруга:** Знижена до 1.1В для підвищення енергоефективності.
* **Архітектура:** Два незалежних 32-бітних підканали на один модуль (забезпечує кращу ефективність доступу до даних).
* **PMIC (Power Management IC):** Контролер живлення вбудовано безпосередньо в модуль пам'яті, що покращує стабільність сигналу.

### Візуальне порівняння (Фізичні відмінності)
Головна фізична відмінність — розташування ключа (вирізу) на контактах. У DDR5 він зміщений ближче до центру, тому модулі **не сумісні** між собою.

### Таблиця порівняння стандартів

| Характеристика | DDR4 | DDR5 |
| :--- | :--- | :--- |
| **Ефективна частота** | 2133 – 3200 МГц (базова) | 4800 – 6400 МГц (базова) |
| **Напруга (V)** | 1.2 V | 1.1 V |
| **Керування живленням** | На материнській платі | На модулі (PMIC) |
| **Канали передачі** | 1 x 64-біт | 2 x 32-біт |

## 2. Огляд та порівняння вибраних модулів пам'яті Samsung для ПК

Для аналізу було обрано два модулі пам'яті виробництва Samsung з інтернет-магазину Rozetka. Обидва призначені для настільних комп'ютерів (UDIMM), мають однаковий об'єм (16 ГБ), але відносяться до різних поколінь.

Класична оперативна пам'ять для систем на базі Intel Core 10-12 поколінь або AMD Ryzen 3000-5000 серій. Це надійний OEM-модуль, який часто використовується в готових офісних та домашніх збірках.

* **Тип:** DDR4
* **Форм-фактор:** UDIMM (для настільних ПК)
* **Об'єм:** 16 ГБ
* **Частота:** 3200 МГц
* **Напруга:** 1.2 В
* **Посилання:** [Samsung DDR4 16GB на Rozetka](https://hard.rozetka.com.ua/ua/336338131/p336338131/)

Модуль нового покоління для сучасних платформ (Intel Core 12/13/14, AMD Ryzen 7000/9000). Забезпечує значно вищу базову швидкість передачі даних прямо "з коробки".

* **Тип:** DDR5
* **Форм-фактор:** UDIMM (для настільних ПК)
* **Об'єм:** 16 ГБ

### Висновок
Порівняння ідентичних за об'ємом планок Samsung демонструє вартість переходу на нові технології:

1.  **Продуктивність:** Перехід на DDR5 дає негайний приріст частоти у 1.5 рази (з 3200 до 4800 МГц), що критично важливо для професійних задач (архівація, рендеринг) та нових ігор.
2.  **Вартість:** За нові технології доводиться платити. Модуль DDR5 коштує майже на 3000 грн дорожче за свого попередника DDR4 (різниця становить близько 61%).
3.  **Рекомендація:** Якщо ви збираєте новий ПК з нуля на сучасній платформі (AM5 або LGA1700), варто переплатити за DDR5 заради перспективи. Для бюджетних збірок DDR4 все ще залишається оптимальним вибором по співвідношенню ціна/якість.
